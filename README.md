# Personal Portfolio Website

A responsive React portfolio website for a Computer Science student.

## Features

- React functional components with hooks
- Modern JavaScript (ES6+)
- Responsive design for mobile, tablet, desktop
- Smooth scrolling and hover animations
- Reusable components for Navbar, projects, certificates, and contact form
- Light / dark theme toggle

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm start
   ```

3. Open the local URL shown in the terminal.

## Project structure

- `src/App.jsx` – main page and content sections
- `src/components/` – reusable UI components
- `src/data/` – sample projects and certificates
- `src/index.css` – global styling
